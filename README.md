# Quiz de Animes

Quiz web con 40 preguntas aleatorias, cada una con imagen del anime. Interfaz con Tailwind CSS e iconos de Lucide; lógica en JavaScript.

## Instalación

```bash
npm install
```

## Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Inicia modo watch (recompila CSS automáticamente) |
| `npm run build` | Compila y minifica CSS para producción |

## Uso

1. Ejecuta `npm run build` para compilar los estilos.
2. Abre `index.html` en el navegador.
3. Selecciona una opción y pulsa "Siguiente" hasta terminar.
4. Al finalizar, verás tu puntaje y podrás "Reintentar".

## Tecnologías

- **Tailwind CSS v4** - Estilos (compilado via CLI)
- **Lucide Icons** - Iconografía
- **JavaScript** - Lógica del quiz
